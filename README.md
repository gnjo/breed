わかりやすさを優先する。
能力値の最低は、XYABxyの中から最高のレベルを一つ抽出する。stable(XYABxy)
祖父母からランダムに値を取る。rmin(XYAB) rmax(XYAB)
直近の親のいずれかをベースとする。base(xy) //現実では祖父母だが。
継代を得ると振れ幅を大きくする。 rate=1+n*0.05

一族全体の到達レベルを組み込む。最終的に全てボーナスとして計上する。
上限は、難易度により変わる。最大難易度はボーナスなし。
素質は常に、最低から最大の二つを保持する。

```
敵の倒し方。倒し方前提とする事で特徴を出す。末尾はヘイズ。
ポイズンヘイズ：ドットダメージを加え敵を倒す。効果は低くとも、重ねがけで加算するように。
スタックヘイズ：ダメージを倍加する。命中率補正は一割ずつ減らす。
マギアヘイズ：魔技を減らし、物理攻撃以外を行えなくする。ドット魔技ダメージ。
デタックヘイズ：相手の物理攻撃力を魔法攻撃力に映す。
トラップヘイズ：在る一定値のダメージを受けた場合。超過分を相手に返す。重ねがけで一定値のボーダーが下がる。
```
